The image tag

The <img> tag is used to display an image in an html.The attributes used are
the attribute src(source) and the attribute alt.

The image tag has several attributes out of which only src and alt are required.The rest are useful but optional attributes.

The source tag

The <src> attribute from the image tag states from where to fetch the image.There are 2 two differents types of URLs you can give for source.

  1. Path to an image file within you web directory site.
  2. Path to an image file that resides elsewhere on the web.

The alt tag

The alt attribute stands for alternate text for an image.

Using this attribue, you can provide a short description of what the image is about. This description should convey information about the image
or its function in the page. The alt is an important attribute because it is the text alternative to the image for users who are unable to see the
image, instead using assistive technology like screen readers that rely on the alt text. It is also useful to provide relevant information for search engines.

Importance of the alt attribute:

  • if you add alt, Screen readers will typically announce that there is an image and read out the contents of the alt attribute.
  • when image is not displayed due to error, it will display the alt text
    • Search engines do not 'see' images and rely on alt and optimize the search.
  • To consume less data, some mobiles users turn off images and make use of alt to get info.

The alt also contributes to semantic meaning - it offers meaning to the image and suggests the prupose of the image content

Purpose of image

You can use images for various reasons in your web page like:

  1. represent a concept, illustration or just a photograph that provide information.
  2. background for a button or link
  3. display a quote or message in the form of text in an image
  4. decorative images

The title, height, width attribute

The title attribute is a global attribute that is used if you ahave a complicated image
that could use a tooltip or description and shows a hiiden message while the cursor hover on the image.

Height

The height attribute is size of an image in vertical dimension. It is used specify the height of an image
by giving it an value.

Width

The width attribute is size of an image in horizantal dimension.It is used to speify the width of an image by giving it an value.

If an image needs to be modify by CSS. it does not need to put in the html.
