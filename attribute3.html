• HTML Element <area>

The area element represents either a hyperlink with some text and a corresponding area on an image map, or a dead area on an image map.
An area element with a parent node must have a map element ancestor or a template element ancestor

List Of Supported attributes for the <area> tag :

global attributes

  1. alt— Replacement text for use when images are not available
  2. coords— Coordinates for the shape to be created in an image map
  3. shape— The kind of shape to be created in an image map
  4. href— Address of the hyperlink
  5. target—Browsing context for hyperlink navigation
  6. download— Whether to download the resource instead of navigating to it, and its file name if so
  7. ping—URLs to ping
  8. rel— Relationship between the location in the document containing the hyperlink and the destination resource
  9. referrerpolicy—Referrer policy for fetches initiated by the element
